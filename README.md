"# nittei-bot" 
 日程調整BOT（LINE連携 + Web）

サークル練習の日程調整を、LINEとWebページで完結させるツール。

できること（現状）
LINE連携: LINEで「日調」と送ると、幹事用ページURLを自動返信します。
自動枠生成: 幹事ページで条件入力すると、練習時間（平日・休日別）や除外時間を考慮したメンバー回答用URLを生成します。
レスポンシブ対応: メンバーはスマホから◯/△/✕で簡単に回答でき、名前入力も必須にしています。
URL共有: 生成されたURLをLINEグループに貼るだけで、すぐに調整を開始できます。

使用技術
Python / Flask(バックエンド)
LINE Messaging API (Bot連携)
JavaScript / Jinja2 (フロントエンド)
git / GitHub*(バージョン管理)

こだわりポイント
セキュリティ: 環境変数（.env）を適切に管理し、機密情報の流出を防止。
UI/UX: 回答者が直感的に操作できるデザインをCSSで構築。

未実装（今後の課題）
データの永続化: 現在はメモリ上での処理のため、DB（SQLite等）の導入を検討中。
集計結果の保存: 回答後の集計結果ページの反映ロジックを調整中。
